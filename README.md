# 本篇为C/C++的SOCKET编程学习    
## Platform：WINDOWS OS ，调用的头文件和方法将会与Linux OS下不同。    
---
前置：需要导入Ws2_32.lib 库
### 服务器端
1.初始化WSADATA  
2.创建套接字(Socket)  
3.初始化Socket内容，也就是service内容，包括地址，端口之类的  
4.绑定端口  
5.申请缓冲区，为后续的消息传输作准备  
6.开始监听端口，等待客户端的访问  
7.结束通讯，释放Socket和WSA数据  

### 客户端
1.初始化WSADATA  
2.创建套接字(Socket)  
3.初始化Socket内容，先初始化客户端自己的，再初始化服务端的，其中服务端的地址大多为变量，等待传入参数    
4.绑定端口  
5.申请缓冲区，为后续的消息传输作准备  
6.链接缓冲区并发送数据，connect函数      
7.结束通讯，释放Socket和WSA数据
